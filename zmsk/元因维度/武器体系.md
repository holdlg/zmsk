
# 武器体系 

## 工艺分类

| 名称  | 周期 | 介绍 |
| ---  | --- | --- |
| 玩具武器 | 一周期 | 玩具类，观赏性，趣味性 |
| 教学武器 | 二周期 | 教育类，模仿真实，使用难度低，杀伤力低 |
| 普通武器 | 三周期 | 家用类，实用，菜刀，剪刀，叉子，筷子，绣花针，铁锅，锄头，耙子，农具等 |
| 民间武器 | 四周期 | 械斗类，刀枪剑戟斧钺钩叉盾，数量一般难成规模 |
| 工业武器 | 五周期 | 枪，炮，装甲，舰艇，飞机，导弹，化学武器，生物武器 |
| 核能武器 | 六周期 | 原子弹，氢弹，聚变，裂变 |
